# 中文字体包下载

**在使用中文模板制作 Mod 之前，请先下载中文字体包。**

> 请注意：我们仍然建议您从各个字体的官网下载这些字体，即便它们都是免费商用字体。

您可以下载懒人专用字体包，下载完后将字体解压到 `game/mod_assets/font` 目录下即可。

当前版本字体包：v2.0

对应支持 v2.0-dev 及旧版的 v1.4.0 版本。
